# 欢迎来到OpenST wiki!  
我们欢迎任何人提交文章，亦或者共同协作其他人的文章  
以下是wiki规范：
> 图片规范：
1. 图片尽量用webp，jpg格式，避免png格式
2. 文章携带图片最大容量50mb
3. 外部引用图片请确保国内外都能正常访问

> 文章规范：
1. 服务器宣传等是不被允许的，这是wiki不是广告
2. 一级标题 # 标题 等使用请避免
3. 文字尽量简洁，中立。政治理念/特殊观点是不被允许的
4. 允许附带其他链接（如外部视频引入等等），但是不能违反文本规范第一条

> 投稿/修改：
1. 审核会在2-3天内完成文章审核
  - 如文章没任何结构/语法，或者事实上的问题，过审核的概率会大很多
  - 如文章有任何问题，将回复未过审核并给予原因。当然可以再次提交审核
  - 如果你对审核的未过审核回复有任何异议，请在issue/q群@staff申诉，并详细写下申诉原因
