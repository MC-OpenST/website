# 📚 欢迎来到OpenST wiki!  
我们欢迎任何人提交文章，亦或者共同协作其他人的文章  
以下是wiki规范：
> 🖼️ 图片规范：
1. 图片尽量用webp，jpg格式，避免png格式
  - 普通照片/截图： 请使用 .jpg（体积最小）。
  - 需要透明背景（如图标，LOGO）： 请务必使用 .webp（比 PNG 更小，加载更快）。
  - 尽量避免使用 .png： 除非你无法转换格式，否则请优先转为 WebP 以节省服务器流量。
2. 单篇文章内所有图片总大小不超过 50MB
3. 外部引用图片无文件后缀限制，只需请确保国内外都能正常访问

> ✍️ 文章规范：
1. 服务器宣传，引流等是不被允许的，这是wiki不是广告
2. 一级标题 # 标题 等使用请避免
3. 文字尽量简洁，中立。请保持内容技术中立，避免涉及政治、宗教或争议性话题
4. 允许附带其他链接（如外部视频引入等等），但是不能违反文本规范第一条

> 📤 投稿/修改：
1. 审核会在2-3工作日内完成文章审核，高峰期可能略有延迟
  - 如果文章结构清晰、语法正确、事实准确，通过审核的概率会更高
  - 如文章有任何问题，将回复未过审核并给予原因。当然可以再次提交审核
  - 如果你对审核的未过审核回复有任何异议，请在issue/q群@staff申诉，并详细写下申诉原因
2. 文章投稿/修改的硬性要求是一个github账号。[点击注册 GitHub 账号](https://github.com/signup)
